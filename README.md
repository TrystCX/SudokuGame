# SudokuGame（网页数独）

一个数独游戏：包含 8000 个关卡（4 种难度），移动端适配。

## 目录结构

- `SudokuGame/`：游戏本体（静态网页）
  - `index.html`：主入口
  - `styles.css`：样式与主题
  - `app.js`：核心逻辑（渲染、交互、存档、设置）
  - `levels.js`：关卡数据
- `数独游戏auto/`：娱乐/测试版（静态网页）
  - 基于 `SudokuGame/` 移植，用于稳定性测试（自动游玩）

## 功能概览

- 计时与最短用时：通关后记录最短用时
- 笔记系统：单格笔记、同数字笔记高亮、选中格笔记联动底部数字键盘
- 一键笔记：自动为所有空格生成候选笔记（覆盖更新）
- 一键擦除笔记：执行“一键笔记”后可一键清空全盘笔记（随后自动恢复普通擦除）
- 主题与配色：
  - 预设配色：绿色 / 蓝色 / 橙色
  - 自定义调色（开发工具）：可逐项微调主题变量（含透明度），支持导入/导出与一键重置；以悬浮窗形式显示，可拖动位置
- 提示系统（Beta）：提供分步提示与可视化高亮；涉及提示相关的功能当前均处于 Beta
  - 技巧分级与支持情况：见 `SudokuGame/TECHNIQUES.md`
  - SukakuExplainer 对齐路线：见 `SudokuGame/SUKAKU_MIGRATION.md`
  - 强制推理提示开关：可在设置中关闭/开启（同时影响 Forcing Cell 与 Nishio）
- 推演（困难/极限）：部分提示支持推演过程抽屉；Nishio 提示提供双分支推演，并支持“简洁/详细”切换与一键跳到矛盾
- 关卡页快捷入口：关卡选择页左上角新增“主页”按钮，可一键返回难度选择页
- 纠错提示：填错数字会标红并提示音；
- 暂停菜单：继续 / 重玩本局 / 设置 / 退出本局
- 设置项：
  - 音效
  - 界面亮度
  - 突出显示区域：在选择一个格子后，高亮显示它所在的九宫、行、列
  - 突出相同数字：在选择一个格子后，如果他包含已经填充的数字，高亮显示数字相同的其他格子
  - 突出相同笔记（数字模式）：在选择一个格子后，如果他包含已经填充的数字，高亮显示数字相同笔记所占的其他格子（数字模式下，高亮显示数字本身）
  - 高亮唯一数：当一个数已经被填充完，那么淡化其数字按钮；在选择一个空格后，如果它所在的九宫、行、或列任意一个中只能填入一个数字，那么淡化其他颜色的数字按钮
  - 选数优先等：开启选数优先后，可以先选择要填充的数字，再点击格子直接填充，如果点击已填充的数字格，那么将填充候选数改为该数字（以便观察与操作），也可在笔记模式下使用，便于快速填写笔记
  - 显示强制推理提示：关闭后不再展示 Forcing Cell 与 Nishio

## 数独游戏auto（自动游玩测试版）

`数独游戏auto/` 用于快速跑关卡稳定性与提示系统稳定性：

- 游戏界面新增“自动”按钮：自动执行“提示 → 下一步 → 应用”循环
- 困难/极限：开局会先执行一次“一键笔记”，再进入自动解题
- 通关后自动进入下一关，直到关卡末尾停止

## 键盘操作（提示/推演）

- 提示面板打开时：W/A/↑/← 切到上一步，S/D/↓/→ 切到下一步（不再移动棋盘选中）
- 推演抽屉打开时：方向键/WASD用于推演步骤导航；当前步骤的“填入/假设”焦点格会自动选中并以蓝底白字强调


## 本地运行（开发/调试）


在仓库根目录启动：

```bash
python -m http.server 5173
```

浏览器访问：

- 游戏本体：`http://localhost:5173/SudokuGame/`
- 自动游玩测试版：`http://localhost:5173/数独游戏auto/`

## 开发模式与开发工具

- 进入开发模式：在游戏地址后加 `?dev=1`（例如 `http://localhost:5173/SudokuGame/?dev=1`），会解锁“开发模式/开发工具”相关开关与入口
- 开发工具：
  - 载入指定盘面、运行提示、搜索示例等
  - 自定义调色：在开发工具中打开“自定义调色”悬浮窗，可拖动并实时预览

## 部署（GitHub Pages）

1. 将整个仓库推送到 GitHub（保持 `SudokuGame/` 位于仓库根目录）
2. 仓库 `Settings → Pages`
3. `Deploy from a branch`，选择 `main`（或 `master`）与 `/ (root)`
4. 部署完成后即可通过 HTTPS 访问：
   - `https://<用户名>.github.io/<仓库名>/SudokuGame/`

## 数据存储

游戏进度、设置与进行中的对局存储在浏览器 `localStorage` 中。清空浏览器站点数据会清除存档。
